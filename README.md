# FHNW prog1 Mühli Game

This repo contains an implementation of "Mühli", a classical board game.

The code was created as part of the prog1 module @ FHNW.

## Releasing a new version

Head to the "Actions" tab, click on the "Release project" workflow.  Enter the version you want to release as version and dispatch a run.

This will release a new jar with the given version and increment the developementVersion to the next possible one.
